# Movie Munchers Frontend Organizational Scheme

This folder defines the base UI and frontend architecture for the Movie Munchers web app.

## Structure
- `components/` – Reusable React components
- `styles/` – Shared CSS variables and themes
- `utils/` – Helper functions
- `context/` – React Context providers
- `pages/` – Page structure for UI testing

This serves as the foundation for the frontend design and integration with the main Movie Munchers app.
